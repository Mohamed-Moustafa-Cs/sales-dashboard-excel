# 📊 Sales Dashboard (Excel)

## 📌 Overview
This project is an interactive sales dashboard built using Microsoft Excel.

## 🛠 Tools Used
- Excel
- Pivot Tables
- Data Visualization

## 📊 Features
- Dynamic filtering (Quarter, Region, Month)
- KPI metrics (Total Sales, Profit, Avg Sales, Customers)
- Interactive charts

## 📈 Insights
- Sales increased from Quarter 1 to Quarter 3
- Some regions generate higher revenue than others
- Strong profit margins observed
